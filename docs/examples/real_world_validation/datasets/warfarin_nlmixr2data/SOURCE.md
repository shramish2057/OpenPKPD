# warfarin provenance

This dataset is `warfarin` from the R package `nlmixr2data`.

Format:
- 519 rows, 9 columns
- id (n=32), time (h), amt (mg), dv (warfarin concentration mg/L or PCA measurement),
  dvid (endpoint identifier), evid, wt (kg), age (yr), sex (factor)

Source reference noted in dataset docs:
Funaki T, Holford N, Fujita S (2018). Population PKPD analysis using nlmixr2 and NONMEM.
